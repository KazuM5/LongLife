Usability Report
•	Efectividad: que los usuarios puedan lograr sus objetivos en nuestra plataforma
•	Eficiencia: que los usuarios pueden hacer sus acciones rápidamente
•	Fácil de aprender: los usuarios usan nuestra plataforma de manera intuitiva 
•	Memorizable: los usuarios usan sin problemas nuestra plataforma después de un tiempo
•	Manejo de errores: la plataforma permite a los usuarios conocer sobre lo errores de manera simple 
•	Satisfacción del usuario: el usuario queda satisfecho con nuestra plataforma y puede tener un control adecuado de las opciones que quiere.
Web Landing Page
-	Esta diseñado bajo el enfoque de Diseño Web Responsive ya que se adapta al dispositivo que está utilizando el usuario.
-	Typography: Se utilizo las recomendaciones dadas.
-	Responsive Images: Se utilizo formatos jpg y png.
-	Trends: Beatiful Animations, HD Background, etc.
-	Patrones de Diseño Web: Pattern Z.
-	Color: Fondo blanco con Intermedio entre Verde (Nature, Safety) y Amarillo (Intellect, Energy, Attention,etc) con una tonalidad suave para que no se vea intenso y sea agradable para la experiencia del usuario.
